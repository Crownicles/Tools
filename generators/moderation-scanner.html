🐾 Noms de Familiers
Colle le résultat de la requête SQL ci-dessous. Le JSON doit être un tableau d'objets avec les champs petEntityId, nickname, ownerKeycloakId.
SELECT pe.id AS petEntityId, pe.nickname, p.keycloakId AS ownerKeycloakId FROM game_PetEntities pe JOIN game_Players p ON p.petId = pe.id WHERE pe.nickname IS NOT NULL AND pe.nickname != '' ORDER BY pe.id;


⚔️ Noms de Guildes
Champs attendus : guildId, name, chiefKeycloakId, elderKeycloakId (nullable).
SELECT g.id AS guildId, g.name, pc.keycloakId AS chiefKeycloakId, pe.keycloakId AS elderKeycloakId FROM game_Guilds g JOIN game_Players pc ON pc.id = g.chiefId LEFT JOIN game_Players pe ON pe.id = g.elderId WHERE g.name IS NOT NULL ORDER BY g.id;


📜 Descriptions de Guildes
Champs attendus : guildId, guildName, guildDescription, chiefKeycloakId.
SELECT g.id AS guildId, g.name AS guildName, g.guildDescription, p.keycloakId AS chiefKeycloakId FROM game_Guilds g JOIN game_Players p ON p.id = g.chiefId WHERE g.guildDescription IS NOT NULL AND g.guildDescription != '' ORDER BY g.id;
